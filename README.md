# endline-bracket

ENDLINE 정모 대진 편성기 — 단독 정적 페이지 (GitHub Pages 호스팅).

- `index.html` : 편성기 한 장 (CDN 폰트 + SheetJS만 외부 의존, 그 외 자급자족)
- `.nojekyll` : Jekyll 처리 우회

## 쓰는 법
1. 명단 입력 — 둘 중 아무거나:
   - **⬆ 엑셀 불러오기** (열: `이름·팀명·성별·나이·등급·초청·게스트`)
   - **✎ 명단 편집** 팝업에서 한 명씩 추가 (팀명/게스트/초청 지정)
   - 명단은 **브라우저 안에서만** 처리됨. **✎ 명단 편집**에서 팀별 **로고 이미지**도 올릴 수 있음.
2. **⚙ 설정** 패널에서:
   - **초청선수**: 전담 탭 기준이 될 선수 토글 (기본=등급 S)
   - **파트너**: 대회 연습용 고정 페어 지정
   - **생성할 탭**: 둘 다 / 선수별 전담 / 빡겜 / 즐겜 / 혼복 / 남복 / 여복 / 파트너전 중 다중 선택
3. **⟳ 탭 생성** → 고른 기준대로 탭이 만들어짐.

## 개인정보
배포본 `index.html`에는 **실명 명단이 없음**(`PLAYERS=[]`). 명단·편성·설정은 그 기기 **localStorage에 자동 저장**되어 새로고침해도 유지되고, 다른 사람/기기에는 빈 화면만 보인다. 「저장본 내보내기」 HTML에는 실명이 들어가므로 repo에 올리지 말 것(`.gitignore`로 차단).

## 로컬 미리보기 / 배포
```bash
cd ~/endline-bracket && python3 -m http.server 8080   # http://localhost:8080
```
`main`에 push하면 GitHub Pages 자동 반영. 새 회차는 엑셀만 새로 올리면 됨(코드·URL 그대로).
